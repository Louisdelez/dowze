## Note sur les sources

Toute la documentation distingue trois niveaux de fiabilité :

- ✅ **Source primaire / solide** : étude évaluée par les pairs, méta-analyse, donnée officielle (ITU,
  W3C, HBS, OCDE…).
- 🟡 **Émergent** : preprint non encore relu, standard récent peu adopté, résultat à confirmer.
- ⚠️ **Auto-déclaré / marketing** : chiffre de fournisseur en conflit d'intérêt — signalé comme tel,
  jamais cité comme preuve.

Cette discipline est un [principe fondateur](../00-FONDATIONS/03-principes-fondateurs.md) (honnêteté).
