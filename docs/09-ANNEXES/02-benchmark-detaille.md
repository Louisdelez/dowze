## Fiches détaillées

### MOOC (Coursera, edX)
- **Force** : savoir d'excellence, gratuit, mondial.
- **Résultats réels** : achèvement médian **12,6 %** (Jordan) ; ~5 % de certifiés (edX) ; taux en
  **baisse** 2013-2018 ; sert surtout les déjà-éduqués (Reich 2019).
- **Limite structurelle** : liberté sans cadre, sans lien, sans finalité.
- **Dowze retient** : la couche humaine (Cercles/Guildes) et la finalité réelle (Quêtes) sont
  indispensables.

### IA conversationnelle seule
- **Force** : personnalisée, patiente, 24/7.
- **Résultats réels** : **−17 %** d'apprentissage durable sans garde-fous (Bastani) ; délestage cognitif
  (MIT, Microsoft, Gerlich).
- **Limite** : illusion de compétence, érosion de l'esprit critique.
- **Dowze retient** : encadrer l'IA (scaffold-and-fade), valider sans elle.

### Khan Academy / Khanmigo
- **Force** : maîtrise + rythme personnel ; déploiement à grande échelle.
- **Résultats réels** : efficacité de Khanmigo **non prouvée par RCT** (chiffres d'adoption seulement).
- **Limite** : arrimé au programme scolaire ; pas un système de vie entière.
- **Dowze retient** : bon socle de maîtrise, mais il faut le sortir du cadre scolaire et le prouver.

### OLPC
- **Force** : accès matériel massif.
- **Résultats réels** : **aucun effet** maths/lecture (Pérou) ; >50 % des machines HS après 3 ans
  (Paraguay).
- **Limite** : l'accès sans pédagogie ni accompagnement ne produit rien ; coûts récurrents
  sous-estimés.
- **Dowze retient** : l'accès ne suffit jamais (leçon n°1) ; Plan Ceibal (Uruguay) a mieux marché grâce à
  l'accompagnement.

### Montessori
- **Force** : autonomie *guidée* ; **preuve la plus solide** (méta-analyse Campbell 2023).
- **Limite** : modèle physique, peu scalable tel quel.
- **Dowze retient** : structure + autonomie guidée = la bonne recette (environnement préparé, matériel
  auto-correctif, travail libre).

### Sudbury / unschooling
- **Force** : motivation intrinsèque, sur-mesure.
- **Limite** : **équité fragile** (dépend du capital familial) ; preuve entachée d'auto-sélection.
- **Dowze retient** : ne pas laisser l'équité au hasard ; l'autonomie totale sans support creuse les
  écarts.

### Micro-écoles / pods (Acton, Sora, Prisma, Synthesis)
- **Force** : petite taille, projets, mentorat, hybride en ligne.
- **Résultats réels** : **quasi aucune preuve** d'efficacité ; coût et sélection.
- **Dowze retient** : design intéressant (projets, mentorat), mais à prouver et à rendre équitable.

### Bootcamps / ISA (BloomTech)
- **Force** : orienté emploi, intensif, par projet.
- **Résultats réels** : placement réel **27-50 %** (vs 85-90 % annoncé) ; **sanctionné par le CFPB**
  (2024) ; ISA prédateurs.
- **Dowze retient** : résultats **auditables et conservateurs** ; jamais d'ISA ni de métriques gonflées.

### École 42
- **Force** : sans prof, P2P, gamifié, gratuit, par projets — le plus proche de Dowze.
- **Résultats réels** : **<30 %** complètent ; « ~100 % d'emploi » **auto-déclaré** (survivants).
- **Limite** : **filtre** au lieu d'inclure ; exclut les non-autonomes.
- **Dowze retient** : pour viser l'universalité, ajouter **scaffolding + soutien** (Mentor + Cercles) à ce
  que 42 laisse à l'auto-sélection.

## La synthèse en une phrase

> Chaque solution a résolu **un** morceau du puzzle. Dowze n'invente pas un nouveau morceau — il invente le
> **plateau** qui les fait tenir ensemble, en corrigeant l'angle mort de chacune (le lien pour les MOOC,
> les garde-fous pour l'IA, l'équité pour 42 et Sudbury, l'honnêteté des métriques pour les bootcamps, la
> pédagogie pour OLPC).
