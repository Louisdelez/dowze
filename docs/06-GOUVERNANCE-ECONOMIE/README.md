## L'enjeu central

> *Et si Dowze réussissait — qui le contrôlerait, et au service de qui ?*

La réponse doit être : **personne ne le contrôle seul, et il sert l'apprenant.** Les échecs étudiés
(BloomTech capturé par le privé → fraude ; LAUSD/AllHere dépendant d'un fournisseur fragile →
effondrement) montrent que la **structure de gouvernance** n'est pas un détail administratif : c'est ce
qui détermine si le système reste digne de confiance.
