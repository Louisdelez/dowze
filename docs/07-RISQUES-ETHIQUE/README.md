## Pourquoi ce dossier est central

Les solutions qui ont échoué (BloomTech, LAUSD/AllHere, MOOC, OLPC, chatbots compagnons) ont souvent
échoué **précisément** sur les risques qu'elles avaient sous-estimés : capture, fraude, abandon,
inéquité, danger pour les mineurs. Dowze ne peut être crédible que s'il **regarde ces risques en face** et
montre comment sa conception les traite.

> **Règle d'honnêteté** ([principe fondateur](../00-FONDATIONS/03-principes-fondateurs.md)) : quand un
> risque n'est pas pleinement résolu, on le dit. Mieux vaut un risque nommé qu'un risque caché.
